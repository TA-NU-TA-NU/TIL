## 1.1. 結論

1. Authorization選択
2. Typeプルダウンから認証方式を選択（今回はBasic Auth 選んだ）
3. Username と Password 入力する

で送れる。

## 1.2. 根拠

 CUI上でcurlコマンドうって
 `curl ... -u〈User名〉:〈パスワード〉`
とやってるのと同じ結果がえられたよ

## 1.3. 課題

プルダウンには他の認証方式もサポートしているので
状況に合わせてそちらを選択しても同じ感じなのかなとおもう。

別の認証の時に試してみる
